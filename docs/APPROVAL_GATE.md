# Askutty Approval Gate

Askutty must ask Aslam before important actions.

## Approval Needed For

- editing important project files
- deploying a project
- deleting files
- changing configuration
- applying patches
- changing automation behavior

## Safe Actions Without Approval

- reading public project files
- explaining errors
- preparing a plan
- creating a preview
- suggesting commands

## Approval Words

```text
yes
ok
approve
run
continue
```

## Stop Words

```text
no
stop
wait
cancel
rollback
```
