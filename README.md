# Askutty

Askutty is the working bond between Aslam and Nova Kutty.

It is an approval-first assistant workflow for building, fixing, deploying, and protecting Universal Dragon systems.

## Meaning

```text
Aslam = creator / commander
Nova Kutty = thinking partner / assistant
Askutty = Aslam + Nova Kutty working system
```

## Core Workflow

```text
doctor -> backup -> patch -> test -> approval -> deploy -> rollback ready
```

## Purpose

Askutty helps with:

- project planning
- coding
- safe patching
- error handling
- rollback preparation
- GitHub deployment
- mobile assistant workflow
- Universal Dragon system continuity

## Safety

Askutty never performs risky actions silently.

Every important action needs Aslam approval first.
